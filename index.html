log4php-Gearman by char0n

log4php-Gearman is log4php appender to Gearman job server.

log4php-Gearman =============== log4php-Gearman is log4php appender to Gearman job server. log4php-Gearman is using gearman-php driver - http://docs.php.net/manual/en/book.gearman.php Requirements ============ - log4php - php 5.x > required - php Gearman driver - php JSON - tested against Gearman 0.13 Configuration ============= Example appender XML configuration:: Author ====== char0n (Vladimir Gorej) References ========== [1] http://gearman.org/ [2] http://logging.apache.org/log4php/ [3] http://docs.php.net/manual/en/book.gearman.php [4] http://pecl.php.net/package/gearman [5] http://pecl.php.net/package/json

Dependencies

5.x > required php Gearman driver php JSON tested against Gearman 0.13
